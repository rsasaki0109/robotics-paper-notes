## TL;DR

- The uncertainty quantification of sensor measurements coupled with deep learning networks is crucial for many robotics systems, especially for safety-critical applications such as self-driving cars.
- This paper develops an uncertainty quantification approach in the context of visual localization for autonomous driving, where locations are selected based on images.
- Key to our approach is to learn the measurement uncertainty using light-weight sensor error model, which maps both image feature and semantic information to 2-dimensional error distribution.
