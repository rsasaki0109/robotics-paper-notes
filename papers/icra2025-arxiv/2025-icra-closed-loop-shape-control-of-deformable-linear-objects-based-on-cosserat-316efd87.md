## TL;DR

- The robotic shape control of deformable linear objects has garnered increasing interest within the robotics community.
- In this work, we present a novel 3D shape control approach that includes the physically realistic Cosserat model into a closedloop control framework, using vision feedback to rectify errors in real-time.
- This is achieved by computing a deformation Jacobian derived from both the Cosserat model and visual data.
