## TL;DR

- Tracking and manipulating irregularly-shaped, previously unseen objects in dynamic environments is important for robotic applications in manufacturing, assembly, and logistics.
- Recently introduced Gaussian Splats efficiently model object geometry, but lack persistent state estimation for task-oriented manipulation.
- We present Persistent Object Gaussian Splat (POGS), a system that embeds semantics, self-supervised visual features, and object grouping features into a compact representation that can be continuously updated to estimate the pose of scanned objects.
