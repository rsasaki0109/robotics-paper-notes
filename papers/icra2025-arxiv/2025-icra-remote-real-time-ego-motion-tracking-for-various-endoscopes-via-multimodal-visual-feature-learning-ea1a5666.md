## TL;DR

- Real-time ego-motion tracking for endoscope is a significant task for efficient navigation and robotic automation of endoscopy.
- In this paper, a novel framework is proposed to perform real-time ego-motion tracking for endoscope.
- Firstly, a multi-modal visual feature learning network is proposed to perform relative pose prediction, in which the motion feature from the optical flow, the scene features and the joint feature from two adjacent observations are all extracted for prediction.
