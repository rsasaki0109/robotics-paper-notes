# robotics-paper-notes

ロボティクス論文（主に2025年）を高速に整理・理解するための、軽量な論文まとめサイトです。実験や実装ではなく、読解・要点整理・自分の解釈に集中します。

## Focus

- SLAM
- Localization
- GNSS
- LiDAR
- Intelligent Vehicles
- Perception
- Mapping
- Sensor Fusion
- Motion Planning
- Control

## Rules

- 1論文 = 1Markdownファイル
- 1論文は60分以内
- 完璧を目指さない
- 自分の解釈を書く
